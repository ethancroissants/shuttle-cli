# ShuttleAI CLI

Autonomous coding agent in your terminal, powered by [ShuttleAI](https://shuttleai.com).

```bash
npm i -g shuttle-ai
shuttle
```

Based on [Cline](https://github.com/cline/cline) (Apache 2.0 © 2026 Cline Bot Inc.)

## Setup

1. Get a free API key at [shuttleai.com/keys](https://shuttleai.com/keys)
2. Run `shuttle` and enter your key on first launch

Or set `SHUTTLEAI_API_KEY` environment variable for headless/CI use.
